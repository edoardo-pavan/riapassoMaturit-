## 1. UNIT 15 — THE WEB

### Web Apps vs Traditional Software

| | Traditional Software | Web App |
|---|---|---|
| **Installation** | Install on each device | Accessed via browser |
| **Updates** | Manual or auto-update | Centralized, instant for all users |
| **Platform** | OS-specific (Windows only) | Cross-platform (works on any browser) |
| **Storage** | Local device | Cloud/remote servers |
| **Examples** | Microsoft Word desktop | Google Docs, Gmail, Figma |

### The Tip of the Iceberg

Most users only see the **surface web** (indexed by Google). Below that:
- **Deep web** (not indexed): private databases, academic journals, email accounts, banking
- **Dark web** (intentionally hidden): requires Tor browser, can host illegal activities

### The Language of the Web

| Acronym | Full Form | What It Is |
|---|---|---|
| **HTML** | HyperText Markup Language | Structure of a webpage |
| **CSS** | Cascading Style Sheets | Styling (colors, layout, fonts) |
| **JS** | JavaScript | Interactivity and dynamic behavior |
| **HTTP/HTTPS** | HyperText Transfer Protocol (Secure) | Protocol for transferring web pages |
| **URL** | Uniform Resource Locator | Web address |
| **DOM** | Document Object Model | Structured representation of a webpage |
| **REST API** | Representational State Transfer | Standard for web APIs |

### The Evolution of the Web

| Version | Era | Characteristics |
|---|---|---|
| **Web 1.0** | 1990s | Read-only, static pages, no interaction |
| **Web 2.0** | 2000s | Read-write, social media, user-generated content, interactive |
| **Web 3.0** | 2010s-now | Semantic web, decentralization, blockchain, AI |
| **Web 4.0** | Emerging | Ubiquitous, symbiotic (human-machine interaction), immersive |

### Creating a Website — Steps

1. **Plan**: define purpose, audience, structure
2. **Design**: wireframes, mockups, UI/UX
3. **Develop**: HTML, CSS, JS (frontend) + backend (ASP.NET Core, Node.js...)
4. **Test**: browsers, devices, performance, accessibility
5. **Deploy**: hosting, domain, SSL certificate
6. **Maintain**: updates, security, content

### SEO — Search Engine Optimization

Techniques to make a website rank higher in search engine results (Google).

- **Keywords**: relevant words in content, titles, meta descriptions
- **Backlinks**: links from other reputable sites → more authority
- **Performance**: fast-loading pages rank better
- **Mobile-friendly**: Google prioritizes mobile-optimized sites
- **Structured data**: helps search engines understand content (schema.org)
- **White hat vs. Black hat**: ethical vs. manipulative SEO techniques

### E-commerce and the Cashless Society

**E-commerce**: buying/selling goods and services online (Amazon, eBay, Shopify).
- **B2C** (Business to Consumer): Amazon
- **B2B** (Business to Business): wholesale platforms
- **C2C** (Consumer to Consumer): eBay, Vinted

**Cashless society**: payments made digitally (credit cards, Apple Pay, PayPal, crypto).
- Pros: convenience, traceability (anti-money laundering), faster transactions
- Cons: privacy loss (every purchase tracked), exclusion of the unbanked, dependence on technology

### Streaming Services

Platforms delivering audio/video in real time (Netflix, Spotify, YouTube, Twitch).
- **On-demand**: user chooses what to watch (Netflix)
- **Live streaming**: real-time broadcast (Twitch, YouTube Live)
- Technology: adaptive bitrate streaming adjusts quality based on connection speed

### Human-Computer Interaction (HCI)

Study of how people interact with computers and designing technology that is intuitive and user-friendly. Key concepts:
- **Usability**: how easy is it to use?
- **Accessibility**: can people with disabilities use it?
- **User experience (UX)**: how does the user FEEL using it?
- **User interface (UI)**: visual elements (buttons, menus, layout)

### Use the Internet Safely

1. **Strong, unique passwords** for each account (use a password manager)
2. **Two-factor authentication (2FA/MFA)** wherever possible
3. **Think before you click**: don't open suspicious links or attachments
4. **Keep software updated**: install security patches
5. **Use HTTPS**: check for the padlock icon in the browser
6. **Be careful what you share** on social media
7. **Use a VPN** on public Wi-Fi

## 2. UNIT 16 — INDUSTRY 4.0 AND THE FUTURE

### The Fourth Industrial Revolution

| Revolution | Era | Key Technology |
|---|---|---|
| **1st** | Late 18th century | Steam power, mechanization |
| **2nd** | Late 19th century | Electricity, mass production (assembly line) |
| **3rd** | Mid 20th century | Computers, automation, electronics |
| **4th** | 2010s-present | IoT, AI, big data, robotics, 3D printing, 5G |

### Foundations of Industry 4.0

| Technology | Role |
|---|---|
| **IoT (Internet of Things)** | Sensors collect data from machines in real time |
| **Big Data & Analytics** | Process massive amounts of data for insights |
| **Cloud Computing** | Store and process data remotely |
| **AI & Machine Learning** | Predictive maintenance, quality control |
| **3D Printing** (Additive Manufacturing) | Produce parts on demand, reduce waste |
| **Augmented Reality (AR)** | Overlay digital info onto the physical world |
| **Autonomous robots** | Work alongside or instead of humans |
| **Digital twins** | Virtual replicas of physical systems for simulation |

### 3D Printing / Additive Manufacturing

Builds objects **layer by layer** from a digital model. Uses plastics, metals, ceramics, even biological materials.
- **Prototyping**: quickly create physical models
- **Custom parts**: medical implants, dental crowns, aerospace components
- **On-demand production**: no need for large inventories

### Drone Delivery

Unmanned aerial vehicles delivering packages. Companies like Amazon (Prime Air), UPS, and DHL are testing it. Benefits: faster delivery, reduced traffic, lower emissions. Challenges: regulations, air traffic safety, weather, weight limits.

### Lasers — How Lasers Are Used

(Light Amplification by Stimulated Emission of Radiation)
- **Manufacturing**: cutting, welding, engraving
- **Medicine**: eye surgery (LASIK), dentistry, dermatology
- **Communications**: fiber optic data transmission
- **Entertainment**: light shows, laser projectors
- **Barcode scanning**: reading QR codes and barcodes

### A Milestone for Artificial Intelligence

Key AI milestones:
- **1997**: IBM Deep Blue beats world chess champion Garry Kasparov
- **2011**: IBM Watson wins Jeopardy!
- **2016**: DeepMind's AlphaGo defeats Go world champion Lee Sedol
- **2022**: ChatGPT released — generative AI goes mainstream
- **2023-24**: AI integrated into search, email, coding, art

### Will Technology Make Humans Redundant?

**Yes (pessimistic view)**:
- Automation replacing jobs (manufacturing, driving, customer service, even coding)
- AI systems performing cognitive tasks better than humans

**No (optimistic view)**:
- New jobs will be created (AI trainers, data ethicists, prompt engineers)
- Technology augments humans, doesn't replace them
- Creativity, empathy, ethical judgment remain uniquely human

### Big Data Is Watching You

Companies collect vast data about you:
- What you search, watch, buy, like
- Your location, habits, social connections
- This data is used for targeted advertising and can be sold

**Concerns**: privacy erosion, manipulation through algorithms, filter bubbles, surveillance.

### The Facebook Whistleblower

**Frances Haugen** (2021): former Facebook employee who leaked internal documents showing Facebook prioritized profits over user safety. Key revelations:
- Instagram harms teenage girls' mental health
- The algorithm promotes divisive content for engagement
- Facebook's anti-hate speech systems were inadequate outside English/US

## 3. GEORGE ORWELL (1903-1950)

### Quick Facts

- **Real name**: Eric Arthur Blair
- **Born**: 1903, Motihari, India (British Empire)
- **Key experiences**: served in the Imperial Police in Burma → rejected imperialism; fought in the Spanish Civil War → opposed totalitarianism
- **Political stance**: democratic socialist, anti-totalitarian (criticized BOTH fascism and Stalinist communism)
- **Famous works**: *Animal Farm* (1945), *Nineteen Eighty-Four* (1949)

### Against Imperialism and Rejection of the Bourgeois

Orwell's experience as a colonial police officer in Burma made him hate imperialism. He saw how the British Empire oppressed local populations. He later lived among the poor in Paris and London (*Down and Out in Paris and London*, 1933) to understand working-class life, rejecting his comfortable bourgeois upbringing.

### Animal Farm (1945)

**Plot**: Farm animals overthrow their human farmer and run the farm themselves, following the principle "All animals are equal." Over time, the pigs (led by Napoleon) become corrupt and gradually change the rules to benefit themselves. The final commandment becomes: *"All animals are equal, but some animals are more equal than others."*

**Allegory**: critique of the Russian Revolution and Stalin's betrayal of socialist ideals. Each animal represents a historical figure (Napoleon = Stalin, Snowball = Trotsky, Boxer = the working class).

### Nineteen Eighty-Four (1949)

**Plot**: Winston Smith lives in a dystopian society ruled by **Big Brother** and the Party, led by the **Thought Police**. The state controls everything: language (Newspeak), history (Ministry of Truth actually destroys and rewrites records), and even thought ("thoughtcrime"). Winston rebels by falling in love with Julia, but they are captured and tortured until he "loves Big Brother."

**Key concepts**:
- **Big Brother**: the all-seeing leader; "Big Brother is watching you"
- **Newspeak**: language designed to limit freedom of thought — if you can't say it, you can't think it
- **Doublethink**: holding two contradictory beliefs at the same time (*"War is Peace, Freedom is Slavery, Ignorance is Strength"*)
- **Telescreens**: devices in every home that both broadcast propaganda AND watch citizens

**Themes**: totalitarianism, surveillance, truth manipulation, loss of individuality, the power of language to control thought

**Famous Quotes**:
- "Who controls the past controls the future: who controls the present controls the past."
- "Big Brother is watching you."
- "War is Peace. Freedom is Slavery. Ignorance is Strength."
